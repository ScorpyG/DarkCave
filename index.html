Projects

- GUESS THE NUMBER -

The program was created using Python coding language. With simple algorithm, repetition and random number generator (RNG) designed for the players to guess the random number within a specific range. After every failed attemps the players will be given hints in order to obtain the correct answer.

The project introduces many different elements and emphasis the basic concepts of programming to beginners. Like buidling code structures, repetition, conditioning, assigning variables, coding standards and navigating through terminal. Simple but yet crucial factors for engineering and software developing era.

- Ping Pong Py -

Ping Pong Py is a 2 player game that was created with Python package called Pygame. The concept is straight forward, there will be two players each player control a paddle using W/S or UP/DOWN arrow key. The objective is to score the ball into eachother goal. If the player scored, the code will trigger a sound function to notify the player that they've scored 1 point. Whoever scored 3 point first win the game (there is an infinite round version to this game as well) ~_^.
