Ping Pong Py

- Ping Pong Py is a 2 player game that was created with Python package called Pygame. The concept is straight forward, there will be two players each player control a paddle using W/S or UP/DOWN arrow key. The objective is to score the ball into eachother goal. If the player scored, the code will trigger a sound function to notify the player that they've scored 1 point. Best of 3 win the game.
